\# LearnToCode 🚀



LearnToCode is a full-stack web application built with ASP.NET Core.  

It is designed as a modern learning platform with authentication, database integration, and scalable architecture.



\---



\## 🌐 Live Site



👉 https://learntocode.runasp.net/



\---



\## ⚙️ Tech Stack



\- ASP.NET Core (.NET 10)

\- C#

\- MySQL

\- Entity Framework Core

\- HTML, CSS, JavaScript

\- Google OAuth Login

\- GitHub OAuth Login



\---



\## 🔐 Features



\- User Registration \& Login system

\- Google Authentication

\- GitHub Authentication

\- Secure session handling

\- MySQL database integration

\- Learning progress tracking system

\- API-ready backend architecture

\- Responsive frontend design

\## 🚀 Deployment



To publish:



dotnet publish -c Release -o publish



Upload the publish folder to /wwwroot on your hosting server.



\---



\## 🔑 Environment Variables



\- Database connection string

\- Google OAuth credentials

\- GitHub OAuth credentials

\- API keys



\---



\## 📦 Setup Locally



git clone https://github.com/yourusername/LearnToCode.git  

cd LearnToCode  

dotnet restore  

dotnet run
